# Accessibility Statement — UC OSPO Education

The UC OSPO Education site is committed to ensuring digital accessibility for people with disabilities. We continually improve the user experience and apply relevant accessibility standards across the site.

## Conformance Status

This site aims to conform to the **Web Content Accessibility Guidelines (WCAG) 2.1 Level AA**. Conformance is self-certified following accessibility testing completed in May 2026, evaluated against the UC IMT-1300 Information Technology Accessibility Policy.

**Testing methods used:**
- Automated scanning with Pa11y (WCAG2AA ruleset) across all pages
- Playwright + axe-core integration tests covering key user flows
- Manual keyboard navigation review
- Color contrast verification
- Semantic structure and ARIA review

## Technical Stack

The site is a static site built with [Astro](https://astro.build), React (for interactive filter components), and plain CSS. It is hosted on GitHub Pages and served at `ucospo.net/education`.

## Contact and Feedback

We welcome accessibility feedback. If you encounter a barrier or cannot access content on this site, please contact us and we will work to provide the information in an alternate format.

- **GitHub:** [Open an issue](https://github.com/UC-OSPO-Network/education/issues/new) in the UC OSPO Education repository
- **Email:** Contact the UC OSPO Network via the [ucospo.net](https://ucospo.net) website

We aim to respond within 5 business days.

## Known Limitations

- Some third-party lesson content is hosted on external sites over which we have no control. We link to these resources but cannot guarantee their accessibility.
- The Keystatic CMS interface (available only in development mode) is not covered by this statement.

## Formal Complaint Process

If you are dissatisfied with our response, you may contact the UCLA Disabilities and Computing Program (DCP) or file a complaint under UC ITAP policy.
